# DMTTY Documentation

This directory contains the documentation for the DMTTY module.

## Contents

- **[dmtty.md](dmtty.md)** - Module overview and architecture
- **[api-reference.md](api-reference.md)** - Complete API documentation

## Viewing Documentation

You can view these documents using `dmf-man`:

```bash
dmf-man dmtty          # Main documentation
dmf-man dmtty api      # API reference
dmf-man dmtty config   # Configuration guide
```
